快速排序

1.排序思路

    快速排序是由冒泡排序改进而得，它的基本思路是：在待排序的n个元素中任取一个元素， 通常取第一个元素作为基准,把该元素放入适当位置后，数据序列被此元素划分成两部分，所有关键字比该元素小的放置在前 一部分，所有关键字比它大的放置在后一部分，并把该元素排在这两部分的中间（称该元素归位），这个过程称作一趟快速排 序，之后对所有划分出来的两部分分别重复上诉过程，直至每部分内只有一个元素或为空为止。简而言之，每趟将表的第一个 元素放入适当位置，将表一分为二，对子表按递归方式继续这种划分，直至划分的子表长为1或。
    一趟快速排序的划分过程是采用从两头向中间扫描的办法，同时交换与基准元素逆序的元素。 具体的做法就是：设两个指针i和j，它们初始时分别指向无序区中第一个和最后一个元素，假设无序区中的元素为R[s],R[s+1] ，...，R[t],则i的初值为s，i的初值为t，首先将R[s]移至变量tmp中作为基准，令j至t起向左扫描直至R[i].key<tmp.key ，将R[j]移至i所指的位置，然后令i自i+1起向右扫描直至R[i].key >tmp.key时，将R[i]移至j所指的位置上，这样重复操作 直至i=j，此时所有R[k](k=s,s+1，...，i-1)的关键字都小于tmp.key，而所有R[k](k=i+1,j+2,...,t)的关键字都大于tmp.key ，此时可将tmp中的元素移至i所指位置R[i]，它将无序中元素分割成R[s..i-1]和R[i+1..t]，再分别进行排序。

2.排序算法

快速排序算法如下：

void QuickSort(RecType R[],int s,int t)	  //对R[s]至R[t]的元素进行快速排序
{
  int i=s,j=t;
  RecType tmp;
  if(s<t)	  //区间内存在至少两个元素的情况
  {
    tmp=R[s];	  //用区间的第一个元素作为基准
    while(i!=j)	  //从区间两段交替向中间扫描，直到i=j为止
    {
      while(j>i && R[j].key>=tmp.key)		
        j--;	  //从右向左扫描，找第1个小于tmp.key的R[j]
      R[i]==R[j];	  //找到这样的R[j],R[i]和R[j]交换
      while(i<j && R[i].key<=tmp.key)
        i++:
      R[j]=R[i];
    }
    R[i]=tmp;
    QuickSort(R,s,i-1);	  //对左区间递归排序
    QuickSort(R.i+1,t);	  //对右区间递归排序
  }
}
