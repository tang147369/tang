顺序查找

1.算法简介

    顺序查找是一种最简单的查找算法。它的基本思路是：从表的一端开始，顺序扫描线性表， 依次将扫描到的关键字和给定k值相比较，若当前扫描到的关键字与k相等，则查找成功；若扫描结束，仍未找到关键字等于k的元素 ，则查找失败。

2.顺序查找算法

顺序查找算法如下(在顺序表R[0..n-1]中查找关键字为k的元素，成功时返回找到的元素的逻辑序号，失败时返回0)

void SeqSearch(SeqList R,int n,KeyType k)	  
{
  int i=0;
  while(i<n && R[i].key!=k)	  //从表头开始往后找
    i++;
  if(i>=n)	  //未找到返回0
    return 0;
  else
    return i+1;	  //找到返回逻辑序号i+1
}
