折半插入排序

1.排序思路

    直接插入排序将无序区中开头元素R[i](i<=i<=n-1)插入到有序区R[0..i-1]中， 此时可以采用折半查找方法在R[0..i-1]中找到插入位置，再通过移动元素进行插入。这样的插入排序称为折半插入排序或二分插入排序。
    在R[low..high](初始时low=0,high=i-1)中采用折半查找方法查找到插入R[i]的位置为R[high+1]，在将R[high+1..i-1] 中元素后移一个位置，并置R[high+1]=R[i]。

2.排序算法

折半插入排序算法如下：

void InsetSort1(RecType R[],int n)	  //对R[0..n-1]按递增有序进行直接插入排序 
{
  int i,j,low,high,mid;
  RecType tmp;
  for(i=1;i<n;i++)
  {
    tmp=R[i];	  //将R[i]保存到tmp中
    low=0;high=i-1;
    while(low<high)	  //在R[low..high]中折半查找有序插入的位置
    {
      mid=(low+high)/2;	  //取中间位置
      if(tmp.keey<R[mid].key)
        high=mid-1;	  //插入点在左半区
      else
        low=mid+1;	  //插入点在右半区
    }
    for(j=i-1;j>=high+1;j--)	  //元素后以
      R[j+1]=R[j];
    R[high+1]=tmp;	  //插入
  }
}
